# ISO2God.app

Simple GUI for converting Xbox/Xbox 360 ISO files to Games on Demand (GOD)
format.  ISO2God.app is a desktop frontend built on top of the Rust library 
[iso2god-rs](https://github.com/iliazeus/iso2god-rs).
